# 🧠 MedAI — AI-Powered Disease Prediction System

## 🚀 About the Project

MedAI is a modern web application that predicts possible diseases based on user-selected symptoms. The goal of this project is to demonstrate how structured logic and intelligent systems can assist users in identifying potential health conditions quickly.

The application focuses on delivering a clean user experience while applying efficient symptom-based prediction logic.

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Language:** TypeScript
* **Styling:** Tailwind CSS

---

## ✨ Key Features

* 🩺 Symptom-based disease prediction
* ⚡ Fast and responsive user interface
* 🧩 Modular and reusable component structure
* 🎯 Simple and intuitive user experience

---

## 📂 Project Structure

```bash
src/
 ├── components/   # UI components
 ├── data/         # Disease dataset
 ├── utils/        # Prediction logic
```

---

## ⚙️ How It Works

Users select symptoms through the interface, and the system analyzes the input against predefined disease data to suggest possible conditions.

---

## 🚀 Getting Started

```bash
npm install
npm run dev
```
